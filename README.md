## Proje Hakkında

## Görev Detayları

### 1. Veritabanı ve SQL
- Platform: Microsoft SQL Server (T-SQL)
- Tablolar: Categories, Products, Customers, Orders, OrderDetails
- Özellikler: JOIN sorguları, GROUP BY analizleri, vw_SiparisOzet view'ı

### 2. Figma UI/UX Tasarımı
- Platform: Figma (Mobil - iOS/Android)
- Ekranlar: Login, Home (Ana Sayfa), Product Detail
- Teknikler: Auto Layout, Component, Variant, 60-30-10 renk kuralı

### 3. Flutter Mobil Uygulama
- Dil: Dart / Flutter
- Özellikler: GridView ürün listesi, JSON veri simülasyonu, Navigator sayfa geçişleri
- Hedef: NovaStore Figma tasarımının birebir kod implementasyonu

### 4. JavaScript Admin Paneli
- Framework: React.js
- CSS: Tailwind CSS
- Özellikler: Ürün ekleme, listeleme, güncelleme, silme (CRUD)
- Yayın: Netlify
