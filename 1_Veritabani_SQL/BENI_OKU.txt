NOVASTORE SQL VERITABANI - KURULUM KILAVUZU
Hazirlayan: Batuhan Tasdemir | Nisan 2026
===================================================================

SQL Server LocalDB kurulumu tamamlandi.
Bilgisayari YENIDEN BASLAT, sonra asagidaki adimlari izle:

-------------------------------------------------------------------
YONTEM 1 - Otomatik (Tavsiye Edilen):
-------------------------------------------------------------------
1. Bilgisayari yeniden baslat
2. Bu klasordeki "SQL_Calistir.bat" dosyasina cift tikla
3. Sonuclar "Sonuc.txt" dosyasina kaydedilir

-------------------------------------------------------------------
YONTEM 2 - Azure Data Studio ile Manuel:
-------------------------------------------------------------------
1. Bilgisayari yeniden baslat
2. Baslat Menusu > "Azure Data Studio" ac
3. "New Connection" tiklayip su bilgileri gir:
   - Connection type: Microsoft SQL Server
   - Server: (localdb)\MSSQLLocalDB
   - Authentication: Windows Authentication
4. Connect > New Query
5. "BatuhanTasdemir_NovaStore_Veritabani.sql" dosyasini ac
6. F5 ile calistir

-------------------------------------------------------------------
YONTEM 3 - Online (Bilgisayar gerekmez):
-------------------------------------------------------------------
1. https://www.db-fiddle.com adresine git
2. Sag ust koseden "Database: SQL Server 2019" sec
3. "BatuhanTasdemir_NovaStore_Veritabani.sql" dosyasinin
   icerigini kopyalayip yapistir
4. "Run" ile calistir
